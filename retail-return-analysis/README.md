# Retail Return Rate Analysis (Tableau)

## Project Overview
This project analyzes product return behavior for a retail company to identify which product categories, regions, and time periods are driving unusually high return rates. The goal of the analysis was to uncover patterns behind returns and provide insights that could help reduce operational losses and improve product strategy.

## Business Problem
Product returns create significant operational costs for retail companies and often indicate deeper issues such as product quality problems, inaccurate product descriptions, logistical challenges, or mismatched customer expectations. Leadership needed better visibility into which categories and regions were contributing most heavily to return activity.

## Data Source
The dataset used in this project was provided within a structured learning environment designed to simulate retail transactional data. While simplified for learning purposes, the dataset includes order records, product categories, return indicators, and geographic information necessary for analyzing return behavior.

## Tools Used
- Tableau Public
- Data Visualization
- Business Analysis

## Analytical Approach
The analysis focused on identifying patterns in return rates across product categories, regions, and time periods. Interactive dashboards were built to allow decision-makers to quickly explore the data and isolate potential drivers behind unusually high return activity.

Key analytical steps included:
- Calculating return rate percentages by category
- Comparing return behavior across geographic regions
- Identifying temporal patterns in return activity
- Visualizing return trends through interactive Tableau dashboards

## Key Insights
The analysis revealed that all three major product categories experienced disproportionately high return rates. Two categories showed return rates approaching double their expected industry benchmarks, while the third category exceeded its standard but to a lesser degree. Technology products created the greatest financial impact due to their higher average price points.

## Business Impact
The analysis identified key drivers contributing to elevated return rates, including geographic concentration in the **West region**, temporal spikes during **Q3 of standard operating years**, and consistently high returns across all three primary product categories. Technology products generated the largest financial impact because of their higher per-unit value. These insights help leadership prioritize investigation into product quality, fulfillment processes, and customer expectations within the most affected segments.
