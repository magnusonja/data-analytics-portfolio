## Featured Project

### Retail Return Rate Analysis (Tableau)

This project analyzes retail product return behavior to identify which product categories, regions, and time periods are driving unusually high return rates.

The analysis revealed that all three major product categories experienced elevated return rates, with two categories approaching double their expected industry benchmarks. The West region and Q3 operating periods showed the highest return concentration, with technology products creating the largest financial impact due to their higher unit cost.

Tools Used: Tableau, Business Analysis, Data Visualization
